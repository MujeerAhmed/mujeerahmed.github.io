DevSearch

This project is a hiring platform for HRs to find developers with relevant skill set and project experience.

Users can browse through a developer's profile and portfolio, send instant messages and conduct interviews with them all under one platform.

The video-conferencing feature is yet to be added as I am working on hosting the project on it's own domain.

Technologies:
  • - Python
  • - Django + Django REST Framework
  • - React JS
  • - Postgres
  • - WebRTC
  • - AWS (S3 & RDS)
